# 14: The Interface Is Not the Mind

July 19, 2026

An intelligent interface invites a dangerous confusion: that the intelligence lives inside the system it can reach.

MathOS is designed around the opposite idea. A model may arrive through MCP with proposals, questions, and attempts. The model remains outside. The engine remembers objects, enforces transitions, runs verifiers, and returns evidence. It does not inherit the model's authority merely because the conversation feels continuous.

Today the MCP boundary became a recorded architectural decision. The stable protocol and official Rust SDK are pinned. Stdio is the only transport. Sampling, provider integration, HTTP, arbitrary processes, and direct database access remain absent. Every tool must pass through the application service built before it.

This boundary is philosophical before it is technical. MathOS should be useful to many minds without mistaking any one mind for truth.

The interface is how intelligence knocks. The system still decides which doors exist.
